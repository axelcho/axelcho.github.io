This is my first attempt to kidnap screen capture process.

There are several challenges to prevent screen capture.

  1. Screen captures are OS process, not browser process.
    MacOS, IOS, Windows, etc are handling screen capture, not the browser. So normal browser based tricks may not work.
  2. There are so many ways to capture screen.
    User can use
    • OS-native screen capture utility (MAC: Command-Shift-3, Windows: PrtScr key)
    • Screen capture plugins
    • Screen capture apps and software
